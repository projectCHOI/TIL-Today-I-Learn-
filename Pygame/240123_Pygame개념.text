1.Pygame이란?
SDL (Simple DirectMedia Layer) 라이브러리를 기반하여,
파이썬을 사용하여 게임을 만들기 위한 크로스 플랫폼 라이브러리

2. SDL (Simple DirectMedia Layer) 라이브러리란?
-C언어
-멀티미디어 처리와 그래픽 출력 기능
@ 사용
-2D 그래픽을 화면에 띄움
-오디오 삽입과 재생
-키보드, 마우스, 게임패드 등 입력장치 기능 활성화
-게임의 진행, 동시 프로제스 수행 같은 물리엔진 기능
-게임 실행 : 윈도우에 설치, 화면 모드 등등..

3. Pygame 사용
-학습용 프로젝트나 소규모 게임 프로젝트에 적합 ex)기초적으로는 체스 게임 만들기 과제 같은거 수행 가능

4.Pygame 설치
파이썬 환경에서 'pip install pygame' 수행 후 코드를 짜내려감.
