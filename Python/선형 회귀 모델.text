@ 선형 회귀 모델이란?
선형 회귀 모델은 통계학에서 사용되는 예측 모델
하나 또는 여러 개의 독립 변수(입력 변수)와 종속 변수(결과 변수) 간의 선형 관계를 모델링하는 방법

@ 단순 선형 회귀
하나의 독립 변수와 하나의 종속 변수 간의 관계를 모델링합니다.[y = ax + b] 형태
y=종속 변수, x=독립 변수, a=기울기, b=y-절편

@ 다중 선형 회귀
두 개 이상의 독립 변수를 사용하여 종속 변수를 예측합니다. [y = a1x1 + a2x2 + ... + anxn + b] 형태
각 x = 다른 독립 변수를(다수), a = 해당 변수의 계수

@ 선형 회귀 모델은 예측값과 실제 값이 ㄷ최대한 동일하게(선형에 근접하게)만들어주는게 포인트
변수를 왕창 줄여야 한다.
이때는 최소제곱법(Ordinary Least Squares, OLS)이라는 기법이 사용된다.

@ 최소제곱법이란?
-1단계 : 잔차의 정의
각 데이터 포인트에 대해, 실제 관측값 y_i와 회귀선에 의해 예측된 값 ax_i + b 사이의 차이를 잔차(residual) e_i라고 합니다. 
즉, e_i = y_i - (ax_i + b)

-2단계 : 잔차 제곱의 합
모든 데이터 포인트에 대해 잔차의 제곱을 계산하고, 이를 모두 더합니다. 
잔차 제곱의 합은 Σ(e_i^2) = Σ(y_i - (ax_i + b))^2입니다.

-3단계 : 최소화 문제
OLS의 목표는 이 잔차 제곱의 합을 최소화하는 기울기 a와 y-절편 b를 찾는 것입니다.

-4단계 : 해의 계산
이 최소화 문제는 보통 미분을 통해 해를 찾습니다. 
a와 b에 대해 잔차 제곱의 합을 미분하고, 이를 0으로 설정하여 최소값을 갖는 a와 b를 찾습니다.
