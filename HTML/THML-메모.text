<문자> </문자> 형식
저 위에 슬레시는 시작과 끝을 나타낸다. 없으면 웹 페이지 작동 안함. ㅜㅜ

<html> </html> 이 안에서 코드를 짠다.
<head> </head> 이 안에서 태그, 문서의 헤더 정보 짠다.
<title> </title> 여기에 웹 페이지의 제목이 뜬다.

<body> </body> 문서에 나타나는 본문 코드를 짠다.
<h1> </h1> 본문에 큰 내용이다.
<p> </p> 본문에 작은 내용이다.

<a> </a> 링크를 넣을 수 있다.
링크 예시 : <a href="https://www.example.com"> </a>
